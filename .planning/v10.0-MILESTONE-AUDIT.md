### Operational Concerns (not code bugs)

**OC-1: migrate-52.ts requires manual execution** (HIGH)
The yield-push endpoint matches on `registry_field_id` + `registry_crop_id` columns added by `scripts/migrate-52.ts`. This standalone script is not in `supabase/migrations/`. If not executed, yield-push silently returns `matched: 0` with no error.

**OC-2: organic-cert hardcodes farm-registry URL** (MEDIUM)
`organic-cert/src/app/(app)/field-enterprises/page.tsx` line 90: `http://localhost:3005` without env override. Works in dev, breaks in non-localhost deployments. All other consumers use proxy routes or env vars.
