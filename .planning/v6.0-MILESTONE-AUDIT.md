## Integration Check

### Connected (18 exports properly wired)

All FSA routes (Phase 27) consumed by FSA UI (Phase 28). Insurance CRUD routes (Phase 29/30) consumed by InsuranceWorkspace. Claims schema (Phase 31) ready for Phase 32.

### Missing Connections (2)

1. **Phase 29 → Phase 30:** `GET /api/insurance/aph-lookup` has no UI caller in InsuranceWorkspace (INS-05)
2. **Phase 29 → Phase 30:** `POST /api/insurance/yield-sync` has no UI trigger in InsuranceWorkspace (INS-06)

### Complete E2E Flows (6)

1. FSA CLU record display (FSA-01, FSA-02)
2. CLU inline edit + bulk operations (FSA-03, FSA-04)
3. Validation warnings on CLU cards (FSA-05)
4. Auto-populate crop from farm-budget (FSA-06)
5. PDF and CSV export (FSA-07, FSA-08)
6. Insurance CRUD + Coverage Matrix + Payout Simulator (INS-01..04, INS-08)

### Broken E2E Flows (4)

1. **APH auto-population:** No fetch to `/api/insurance/aph-lookup` from any UI component
2. **Yield sync from grain-tickets:** No "Sync Yield" button in InsuranceWorkspace
3. **Create claim from policy:** No UI button to call `POST /api/claims`
4. **Claims Kanban pipeline:** Phase 32 not built
