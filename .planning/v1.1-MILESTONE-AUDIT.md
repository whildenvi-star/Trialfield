### Integration Issues

**INT-01 (Medium): Fallow enterprise edit loses cost fields**

`field-enterprises/page.tsx` `openEdit()` resets `fallowCostAmount` and `fallowCostCategory` to empty string instead of reading existing values. The `FieldEnterprise` interface omits these fields. Saving an edit silently nullifies stored cost data.

- **Affects:** SCHEMA-03, VIEW-05
- **Fix:** Add `fallowCostAmount: number | null` and `fallowCostCategory: string | null` to `FieldEnterprise` interface; pre-populate in `openEdit()`
- **Severity:** Medium — data safety issue but only affects edit path for fallow enterprises
