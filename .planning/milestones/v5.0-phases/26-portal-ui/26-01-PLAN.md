    - `type: 'default'` (bezier curves)

    **React Flow config:**
    - `fitView` enabled so the map auto-fits the viewport
    - `proOptions={{ hideAttribution: true }}` to remove the React Flow watermark
    - `nodesDraggable={false}` — static layout, no dragging
    - `nodesConnectable={false}` — no user connections
    - `zoomOnScroll={false}`, `panOnDrag={false}` — static viewing experience
    - Background: transparent (parent div provides soil-bg)
