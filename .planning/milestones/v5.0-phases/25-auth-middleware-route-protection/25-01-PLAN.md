  <files>glomalin-portal/src/app/actions/auth.ts</files>
  <action>
Create `glomalin-portal/src/app/actions/auth.ts` with three server actions:

1. **login(formData: FormData)**: Extract email+password from formData. Call `createClient()` (from `@/lib/supabase/server`) then `supabase.auth.signInWithPassword({ email, password })`. On error, redirect to `/login?error=invalid` (generic — do NOT reveal which field failed per CONTEXT.md security decision). On success, redirect to `/dashboard`.

2. **logout()**: Call `createClient()` then `supabase.auth.signOut()`. Redirect to `/login`.

3. **resetPassword(formData: FormData)**: Extract email. Call `createClient()` then `supabase.auth.resetPasswordForEmail(email, { redirectTo: '${origin}/auth/callback' })`. Redirect to `/forgot-password?sent=true`. Do NOT reveal whether the email exists.

All three must be `'use server'` functions. Use `redirect()` from `next/navigation` for redirects (redirect throws, so call it AFTER the try/catch, not inside). Use `revalidatePath('/', 'layout')` before redirect in login/logout to clear cached data.
  </action>
