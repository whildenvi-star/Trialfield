## Tech Debt (4 items, no blockers)

### Phase 7: All-Enterprise Sync
- **Unguarded sync endpoint** — `POST /api/fields/sync-macro` has no `getAuthContext()` or `hasPermission()` check. Any network caller can trigger a full sync. Operation is additive and idempotent (upsert pattern), so risk is data refresh, not data loss. UI caller is gated by `canSeeBudget`. **Severity: Medium.**

### Phase 8: Farm-Wide Budget Summary
- **Sidebar link visible to CREW** — `budget-summary` nav item in `sidebar.tsx` has no role gate. CREW users see the link, click it, get 403 from API. No data leak (API enforces `budget:read`), but CREW lands on an error page. **Severity: Low.**

### Phase 6: Actuals Entry
- **Seed ActualCell label mismatch** — ActualCell displays `actualPricePerUnit` ($/unit) with `/ac` sub-label. Variance math is correct after 6.1 fix (`actualPricePerUnit * rate`), but the displayed cell value before variance computation is $/unit, not $/acre. **Severity: Low (cosmetic).**

### Documentation
- **Missing SUMMARY frontmatter** — `requirements-completed` field absent from all 10 SUMMARY.md files. 3-source cross-reference relies on VERIFICATION.md + REQUIREMENTS.md only. **Severity: Low (documentation gap).**
