## High-Priority Integration Risks (for Phase 10)

1. **Farm.name vs f.farm field mismatch** — Prisma `Farm.name` but all consumers expect `farm` key. `dbFarmToJson` remapping is critical or farm summaries silently return 0 BU. (DB-02, DB-04)

2. **Service worker CACHE_NAME bump** — ticket ID type changes from string (`t_000001`) to integer. CACHE_NAME must bump atomically with route rewrite or cached PWA passes string IDs to integer routes → all edit/delete breaks. (DB-02, DB-04)

3. **GET /api/crops shape** — CropConfig becomes Prisma array rows but client expects object keyed by crop name. Route must reconstruct object shape. (DB-02, DB-04)
