Establish the Phase 1 data foundation: Prisma schema extensions for staging/mapping/sync-state, TypeScript port of the FieldOps OAuth2 client, mock data for development, and a normalizer that converts raw Case IH API responses into typed staging records validated with Zod.

</objective>
