    Pure async function that reads FieldEnterprise + existing FieldHistory and returns SnapshotPreviewRow[]:
    - `import { prisma } from "@/lib/prisma";`
    - `export async function buildSnapshotPreview(farmId: string, cropYear: number): Promise<SnapshotResult>`
    - Query `prisma.fieldEnterprise.findMany({ where: { field: { farmId }, cropYear }, include: { field: { select: { name: true, organicStatus: true } } } })`
    - Group by fieldId using a Map. For each field's enterprises:
      - If multiple enterprises: concatenate non-fallow crop names with " / " as the crop string; build notes string showing each enterprise's crop + acres
      - If single enterprise: use its crop directly; if isFallow, set crop to "Fallow"

    - Query existing FieldHistory for these fields+year: `prisma.fieldHistory.findMany({ where: { fieldId: { in: [...fieldIds] }, year: cropYear } })`
    - Compare each row against existing: if no existing record -> action='new'; if existing and crop/organicStatus changed -> action='update'; else -> action='unchanged'
    - Return `{ preview: rows, summary: { new, update, unchanged, total: rows.length } }`

    **3. Create POST /api/rotation-snapshot/[year]/take/route.ts:**
    Follow the exact contract from Phase 16/17 compile routes (preview/commit body pattern):
    - Parse year from params (await params, parseInt)
    - Parse body: `{ preview?: boolean }` — default to preview mode (`preview !== false`)
    - Find farm: `prisma.farm.findFirst()` — return 400 if not found
    - Call `buildSnapshotPreview(farm.id, year)` to get rows
    - If preview mode: return `{ preview: rows, summary }` with `Cache-Control: no-store`
    - If commit mode: run `prisma.$transaction()` with array of `prisma.fieldHistory.upsert()` calls:
      - `where: { fieldId_year: { fieldId: row.fieldId, year } }`
      - `create: { fieldId: row.fieldId, year, crop: row.crop, organicStatus: row.organicStatus, notes: row.notes }`
      - `update: { crop: row.crop, organicStatus: row.organicStatus, notes: row.notes }`
    - Only upsert rows where action !== 'unchanged'
    - Return `{ committed: committedCount, total: rows.length }` with `Cache-Control: no-store`
    - Wrap in try/catch, return 500 on error

    - Get current year from query param `?currentYear=2025` or default to current calendar year
    - Calculate year range: `[currentYear, currentYear-1, ..., currentYear-(years-1)]`
    - Find farm: `prisma.farm.findFirst({ include: { fields: { orderBy: { name: 'asc' }, select: { id: true, name: true } } } })`
    - Query FieldHistory: `prisma.fieldHistory.findMany({ where: { year: { in: yearRange }, field: { farmId: farm.id } } })`
    - Build RotationRow[] by iterating fields, filling history Record with year keys (null for years with no snapshot)
    - Return `{ fields: RotationRow[], years: yearRange }`
    - Cache-Control: no-store

    ```typescript
    async function handleTakeSnapshot() {
      if (snapshotStatus?.exists) {
        if (!window.confirm(`A snapshot already exists for ${cropYear}. Replace it?`)) return;
      }
      setSnapshotLoading(true);
      setSnapshotMessage(null);
      try {
        const res = await fetch(`/api/rotation-snapshot/${cropYear}/take`, {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ preview: false }),
        });
        if (!res.ok) throw new Error('Snapshot failed');
        const result = await res.json();
        setSnapshotMessage(`${result.committed} fields snapshotted for ${cropYear}`);
        setSnapshotStatus({ exists: true, fieldCount: result.total });
      } catch (e) {
        setSnapshotMessage('Snapshot failed — check server logs');
      } finally {
        setSnapshotLoading(false);
      }
    }
    ```
