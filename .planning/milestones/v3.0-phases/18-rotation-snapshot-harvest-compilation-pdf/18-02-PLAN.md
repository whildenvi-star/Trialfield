    **`export async function mapHarvest(cropYear: number, farmId: string): Promise<HarvestCompileResult>`**

    Step 1: Fetch tickets from grain-tickets:
    - Call `getTicketsForCropYear(cropYear)` — returns TicketRecord[]
    - If this throws (grain-tickets unavailable), re-throw with a clear message

    Step 2: Get all organic FieldEnterprise records for the farm+year:
    - `prisma.fieldEnterprise.findMany({ where: { field: { farmId }, cropYear }, include: { field: { select: { id: true, name: true, farmBudgetFieldName: true } } } })`
    - Build a field lookup map: lowercased field.name -> { fieldId, fieldName, enterprises[] }, and also lowercased farmBudgetFieldName -> same entry (for alias matching)
    - Build an enterprise lookup: for each enterprise, create a key with `fieldId + normalizedCrop` -> enterprise

    Step 3: Match each ticket to a field:
    - For each ticket: normalize `ticket.farm` to lowercase, look up in field name map, then farmBudgetFieldName map
    - Track matchMethod: 'name' if field.name matched, 'stored-mapping' if farmBudgetFieldName matched
    - If no field match: add to `unmatched[]` with reason='no-field-match'

    Step 4: Match ticket crop to enterprise crop:
    - Normalize ticket.crop using `normalizeCropName(ticket.crop)` — strips "Org " and similar prefixes
    - For the matched field, find an enterprise where `normalizeCropName(enterprise.crop).toLowerCase() === normalizedTicketCrop.toLowerCase()`
    - If no crop match within the field's enterprises: add to `unmatched[]` with reason='no-crop-match'

    Step 5: Group matched tickets by fieldEnterpriseId:
    - Sum totalLoads (count) and totalNetWeight (sum of netWeight)
    - Use the latest ticket.date as harvestDate
    - Use enterprise.plantedAcres as acresHarvested

    Step 6: Compare against existing SYNCED HarvestEvents:
    - Query `prisma.harvestEvent.findMany({ where: { fieldEnterpriseId: { in: matchedEnterpriseIds }, dataSource: 'SYNCED' } })`
    - If existing SYNCED event exists for the enterprise: action='update' if netWeight differs, else 'unchanged'
    - If no existing: action='new'

    Step 7: Consolidate unmatched by (ticketFarm, ticketCrop, reason) — sum counts and weights

    Step 8: Return `{ preview: matchedRows, unmatched: unmatchedRows, summary: { new, update, unchanged, unmatched: unmatchedRows.length } }`

    **Important constraints:**
    - Group by ticket.id (NOT ticketNo) when summing — ticketNo allows duplicates (known schema design)
    - Case-insensitive matching for both field names and crop names
    - Only include tickets for fields that have enterprises in organic-cert (not all grain-ticket deliveries)

    - Parse year from params (await params, parseInt)

      - Return `{ committed: committedCount, unmatched: result.unmatched.length }` with `Cache-Control: no-store`

    ```typescript
    async function handleCompileHarvest() {
      setHarvestLoading(true);
      setHarvestCommitMessage(null);
      try {
        const res = await fetch(`/api/compile/${cropYear}/harvest`, {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ preview: true }),
        });
        if (!res.ok) {
          if (res.status === 503) {
            setHarvestCommitMessage('grain-tickets is unavailable — start it on port 3000');
            return;
          }
          throw new Error('Harvest preview failed');
        }
        setHarvestResult(await res.json());
      } catch (e) {
        setHarvestCommitMessage('Failed to compile harvest data');
      } finally {
        setHarvestLoading(false);
      }
    }
    ```

    **Handler: handleCommitHarvest():**
    ```typescript
    async function handleCommitHarvest() {
      if (!window.confirm(`Commit ${harvestResult?.summary.new ?? 0} new + ${harvestResult?.summary.update ?? 0} updated harvest events?`)) return;
      setHarvestLoading(true);
      try {
        const res = await fetch(`/api/compile/${cropYear}/harvest`, {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ preview: false }),
        });
        if (!res.ok) throw new Error('Commit failed');
        const result = await res.json();
        setHarvestCommitMessage(`Committed ${result.committed} harvest events`);
        // Re-run preview to refresh state
        handleCompileHarvest();
      } catch {
        setHarvestCommitMessage('Commit failed — check server logs');
      } finally {
        setHarvestLoading(false);
      }
    }
    ```
