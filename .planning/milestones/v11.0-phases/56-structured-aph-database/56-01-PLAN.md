  <files>glomalin-portal/scripts/migrate-56.ts, glomalin-portal/src/lib/insurance/calc.ts</files>
  <action>
    **migrate-56.ts** — Create Supabase migration script following the established migrate-29.ts / migrate-52.ts pattern (manual .env parse, exec_sql RPC, verify select):

    CREATE TABLE aph_records:
    - id: uuid DEFAULT gen_random_uuid() PRIMARY KEY
    - policy_id: uuid NOT NULL REFERENCES insurance_policies(id) ON DELETE CASCADE
    - crop_year: integer NOT NULL (the year this yield came from, e.g., 2023)
    - actual_yield: numeric NOT NULL (bu/ac for that year)
    - source: text NOT NULL DEFAULT 'manual' (one of: 'manual', 'grain-tickets', 'import')
    - is_disaster_year: boolean NOT NULL DEFAULT false (excluded from APH calculation when true)
    - notes: text (optional notes about this year's yield)
    - created_at: timestamptz DEFAULT now()
    - updated_at: timestamptz DEFAULT now()
    - UNIQUE(policy_id, crop_year) — one APH record per policy per year

    Add RLS policy: authenticated_read (SELECT for authenticated), authenticated_write (INSERT/UPDATE/DELETE for authenticated).

    Also add an index on policy_id for efficient lookups.

    **calc.ts** — Add two new pure functions (append after the existing computeClaimAlert function):

    1. `computeAphFromRecords(records: AphRecord[]): { aph: number; includedCount: number; excludedCount: number; totalCount: number }`
       - Filter out records where is_disaster_year = true
       - Filter out records where actual_yield <= 0
       - Compute simple average of remaining yields
       - Return aph rounded to 2 decimals, plus counts for UI display
       - If no includable records, return aph: 0

    2. `computeGuarantee(aph: number, coverageLevel: number): number`
       - Returns round2(aph * (coverageLevel / 100))
       - coverageLevel is stored as integer (e.g., 75 = 75%)

    Add the AphRecord interface:
    ```typescript
    export interface AphRecord {
      id: string
      policy_id: string
      crop_year: number
      actual_yield: number
      source: 'manual' | 'grain-tickets' | 'import'
      is_disaster_year: boolean
      notes: string | null
      created_at: string
      updated_at: string
    }
    ```
