    Create position-table.tsx:
    - Receives positions: MarketingPosition[] and onSelectCrop callback
    - Renders a table with columns: Crop | Est. Production (bu) | Contracted (bu) | Unpriced (bu) | CBOT Price | Unpriced Exposure ($)
    - Est. Production comes from grain-tickets yield summaries aggregated by crop (sum totalNetBU across all fields for each registryCropId)
    - Contracted = sum of bushels from all contracts for that crop (across all contract types)
    - Unpriced = Est. Production - Contracted (floor at 0)
    - CBOT Price = matched price from cbot-prices response by crop name mapping
    - Unpriced Exposure = Unpriced bu x CBOT Price (show as formatted dollar amount)
    - Position computation logic: group contracts by crop (using registry_crop_id if available, fall back to crop name string match). Merge with yield summaries by registry_crop_id. For crops with contracts but no production data, show production as "—".
    - Total row at bottom summing contracted bu, unpriced bu, and total exposure

    - Font: font-mono throughout (consistent with portal)
    - Clicking a crop row expands inline to show individual contracts for that crop (mini-table with buyer, type, bushels, price, delivery window)
    - Each contract row has Edit and Delete action links
