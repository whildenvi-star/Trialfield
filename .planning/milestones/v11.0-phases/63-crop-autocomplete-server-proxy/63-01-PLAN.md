<objective>
Replace the hardcoded `http://localhost:3005/api/crops/autocomplete` URL in contract-drawer.tsx with a portal-relative call to a new server-side proxy route.

Purpose: The hardcoded localhost URL silently fails in production — the VPS client browser cannot reach port 3005 directly. Moving the fetch server-side via a portal proxy route resolves the failure without any client network access to farm-registry.

Output: New route at `/api/registry/crops`, updated `contract-drawer.tsx` calling `/api/registry/crops?q=...`.
</objective>

.then((data: { crops?: { name: string }[] } | { name: string }[] | string[]) => {

    setCropSuggestions(data.crops.map((c) => c.name).slice(0, 6))
