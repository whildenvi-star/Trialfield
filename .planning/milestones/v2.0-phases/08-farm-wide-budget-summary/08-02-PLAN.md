**Session and role:**
```typescript
const { data: session } = useSession();

const canSeeFinancial = role === "ADMIN";
```

**Table layout — Macro Rollup replica:**

The table should mirror the Macro Rollup spreadsheet layout. Column structure:

| Enterprise | Field | Acres | Seed P | Seed A | Fert P | Fert A | Chem P | Chem A | Ops P | Ops A | Total P | Total A | Variance | [Rev] | [Margin] |

Where "P" = Projected per-acre, "A" = Actual per-acre. Financial columns [Rev] and [Margin] only render if `canSeeFinancial`.

**Simplify column headers if table is too wide.** Claude's discretion per CONTEXT.md — consider using sub-headers:

```
| Enterprise | Field | Acres | Seed | Fertilizer | Chemical | Operations | Total | Var | [Financial] |
|          |       |       | Proj | Act | Proj | Act | Proj | Act | Proj | Act | Proj | Act |     |             |
```

Or a simpler approach: show only Total Projected, Total Actual, and Total Variance per enterprise for the summary table (since the per-category breakdown is available by clicking into the enterprise detail). Use your judgment for the best balance of information density and readability.

**Section rendering:**

1. **Page header:** "Farm Budget Summary" title with the crop year displayed
2. **Organic section:** Section header "Organic", enterprise rows, subtotal row (styled differently — bold/background)
3. **Conventional section:** Section header "Conventional", enterprise rows, subtotal row
4. **Grand total row:** Visually distinct (darker background, bold)

**Enterprise row rendering:**
- All cost values shown as currency format: `$X.XX`
- Actual columns: show `—` when null (no actuals entered yet)
- Variance column: use green/red color coding. Reuse the `cn` utility with `text-green-700` / `text-red-700` — same pattern as `VarianceCell.tsx`. Favorable (variance >= 0) = green, unfavorable = red.
- Row is clickable: `onClick={() => router.push(\`/field-enterprises/${row.enterpriseId}\`)}` with `cursor-pointer hover:bg-stone-800/50` styling

**Subtotal and grand total row rendering:**
- Same column structure as enterprise rows
- Subtotal: medium font weight, subtle background (`bg-stone-800/30`)
- Grand total: bold, darker background (`bg-stone-800/60`)
- Weighted average values from API response (already pre-computed)

**Financial columns (ADMIN only):**
