1. **fsa-crop-list.ts** — DELETE the hardcoded `FSA_CROP_LIST` array entirely. Replace with an async function that fetches from the registry:
```typescript
export async function fetchCropList(): Promise<string[]> {
  const res = await fetch('/api/registry/crops-autocomplete');
  if (!res.ok) throw new Error('Failed to fetch crop list from registry');
  const crops = await res.json();
  return crops.map((c: { name: string; organic: boolean }) =>
    c.organic ? `Organic ${c.name}` : c.name
  ).sort();
}
```
NOTE: FSA crop names include the organic prefix for display purposes (FSA reporting uses "Organic Corn" not just "Corn"). The registry stores organic as a boolean, so reconstruct the display name.

IMPORTANT: Exclude FSA land-use categories (Idle, Fallow, CRP, Cover Crop) from the crop list. These are NOT in the registry per user decision. If the portal needs these categories for FSA reporting, keep them as a small local constant array SEPARATE from the crop list, merged at the point of use.
