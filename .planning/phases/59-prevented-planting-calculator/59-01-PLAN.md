**PP Indemnity Formula (RMA standard):**
- PP coverage factor: 60% for most crops (corn, soybeans, wheat, barley, oats, rye). This is the standard RMA prevented planting coverage factor.
- PP indemnity = guarantee × pp_coverage_factor × spring_price × prevented_planting_acres
- Where `guarantee` is the APH guarantee (bu/ac), `spring_price` comes from insurance_pricing, and `prevented_planting_acres` is the acres that could not be planted.

Function signature:
```typescript
export const PP_COVERAGE_FACTOR = 0.60  // RMA standard: 60% of guarantee for most crops

export function computePpIndemnity(
  policy: { guarantee: number; prevented_planting_acres: number | null },
  pricing: PricingEntry[],
  crop: string | null
): { ppIndemnity: number; ppGuarantee: number; springPrice: number; ppFactor: number }
```

Implementation:
1. Look up spring_price from pricing array using the existing `findPrice` helper pattern (match by crop name, case-insensitive). Note: `findPrice` is a module-private function — duplicate the lookup inline or extract it. Since findPrice is already in `@/lib/fsa/calc.ts` as a private function, just duplicate the pattern: iterate pricing, match `crop.toLowerCase().trim()`.
2. ppGuarantee = guarantee × PP_COVERAGE_FACTOR
3. ppIndemnity = round2(ppGuarantee × springPrice × (prevented_planting_acres ?? 0))
4. Return { ppIndemnity, ppGuarantee, springPrice, ppFactor: PP_COVERAGE_FACTOR }
5. If no pricing match or no prevented_planting_acres, return zeros.

Export the constant and function so both UI and PDF can use them.
