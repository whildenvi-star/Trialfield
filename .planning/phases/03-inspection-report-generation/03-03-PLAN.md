  <name>Task 1: API Routes (Generate, List, Download)</name>
  <files>
    organic-cert/src/app/api/reports/generate/route.ts

  <action>
    Follow the admin route boilerplate pattern from Phase 1: auth() -> session check -> farmId extraction. All routes require authenticated user with farmId.

    1. **POST /api/reports/generate** (generate/route.ts):
       - Auth check: session.user.farmId required, return 401 if missing.
       - Parse body: { cropYear: number, fieldIds?: string[] }. Validate cropYear is a reasonable integer (2000-2100 range).
       - Call assembleReportData(farmId, cropYear, fieldIds || null) to get ReportData.
       - Import renderToBuffer from "@react-pdf/renderer" and InspectionReport from "@/lib/pdf/inspection-report".
       - Call `const buffer = await renderToBuffer(<InspectionReport data={reportData} />)`.
       - Create uploads directory: `await fs.mkdir(path.join(process.cwd(), "uploads", "reports"), { recursive: true })`.
       - Generate filename: `inspection-${cropYear}-${Date.now()}.pdf`.
       - Write buffer to file: `await fs.writeFile(filePath, buffer)`.
       - Create GeneratedReport record in Prisma: farmId, cropYear, filename, filePath (absolute), fieldCount (reportData.fields.length).
       - Return JSON: { id: report.id, filename, cropYear } with status 201.
       - Wrap entire body in try/catch; on error, return 500 with { error: "Failed to generate report" } and console.error the actual error.

    2. **GET /api/reports** (route.ts):
       - Auth check with farmId.
       - Query `prisma.generatedReport.findMany({ where: { farmId }, orderBy: { createdAt: "desc" } })`.
       - Return JSON array of reports: { id, cropYear, filename, fieldCount, createdAt }.

    3. **GET /api/reports/[id]** ([id]/route.ts):
       - Auth check with farmId.
       - Extract id from `await params` (Next.js 16 pattern: `{ params }: { params: Promise<{ id: string }> }`).
       - Find report: `prisma.generatedReport.findUnique({ where: { id } })`.
       - 404 if not found or report.farmId !== farmId (tenant isolation).
       - Read file: `await fs.readFile(report.filePath)`.
       - Return as NextResponse with headers: Content-Type "application/pdf", Content-Disposition `attachment; filename="${report.filename}"`.
       - If file read fails (file deleted from disk), return 404 with { error: "Report file not found" }.
  </action>
