  <name>Task 2: Report Data Assembler</name>
  <files>organic-cert/src/lib/report-assembler.ts</files>
  <action>
    Create report-assembler.ts that exports:

    1. **ReportData interface** — typed shape for all report data:
       ```typescript
       export interface ReportData {
         farm: FarmInfo;
         cropYear: number;
         generatedAt: Date;
         fields: FieldWithHistory[];
         allApplications: ApplicationRecord[];
         allHarvests: HarvestRecord[];
         massBalance: MassBalanceByCrop[];
       }
       ```
       Define supporting interfaces: FarmInfo (name, operatorName, county, state, certNumber, address, logoPath), FieldWithHistory (field info + 3-year enterprises with all sub-records), ApplicationRecord (flattened: fieldName, date, materialName, rate, rateUnit, acres, nopStatus, applicator), HarvestRecord (flattened: fieldName, date, crop, lotNumber, acresHarvested, yieldPerAcre, yieldUnit, netWeight, equipment, dataSource), MassBalanceByCrop (crop name, totalHarvestedLbs, totalSoldLbs, lots: array of { lotNumber, harvestedLbs, soldLbs, hasSales }).

    2. **assembleReportData(farmId, cropYear, fieldIds?)** — single Prisma query using nested includes:
       - Fetch farm with all fields (or filtered by fieldIds if provided)
       - Include 3 years of enterprises (cropYear, cropYear-1, cropYear-2) with materialUsages (include material), fieldOperations (include equipment), harvestEvents (include cropLots with loadoutEvents and saleDelivery), fertilityEvents
       - Include field history for same 3 years
       - Flatten applications across all fields/enterprises into allApplications array, sorted by date
       - Flatten harvests across all fields/enterprises into allHarvests array, sorted by date. For each harvest, pull lot number from associated CropLot.
       - Compute mass balance: query CropLots for the current cropYear only, group by crop, then by lot. For each lot, sum loadoutEvent.saleDelivery.quantityLbs as soldLbs. Format as MassBalanceByCrop[].
       - Return complete ReportData object.

    Use Prisma client from @/lib/prisma. Use date-fns for any date formatting. Follow tenant isolation pattern (farmId in where clause). Handle null/undefined fields defensively (e.g., field with no enterprises, enterprise with no harvests).
  </action>
