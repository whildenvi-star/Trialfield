**page.tsx** (server component):
Fetch lightweight counts from Supabase for the Overview stats row. Use `CURRENT_CROP_YEAR` from `@/lib/config`. Run these three queries with `Promise.all`:
1. `clu_records` count where `crop_year = CURRENT_CROP_YEAR` and `reported_date IS NULL` → `unreportedCount`
2. `insurance_policies` count where `policy_year = CURRENT_CROP_YEAR` → `activePoliciesCount`
3. `claims` count where `stage NOT IN ('closed', 'denied')` → `openClaimsCount`

Use `.select('id', { count: 'exact', head: true })` for all three (no data fetch, just counts).

Pass props to `ComplianceShell`:
```tsx
<ComplianceShell
  unreportedCount={unreportedCount ?? 0}
  activePoliciesCount={activePoliciesCount ?? 0}
  openClaimsCount={openClaimsCount ?? 0}
/>
```

Wrap in `<Suspense fallback={null}>` because ComplianceShell uses `useSearchParams()`.

**compliance-shell.tsx** (client component, `'use client'`):
Props: `{ unreportedCount: number, activePoliciesCount: number, openClaimsCount: number }`

Tab definition type:
```ts
type TabId = 'overview' | 'acreage' | 'insurance' | 'claims' | 'calendar'
const TABS: { id: TabId; label: string }[] = [
  { id: 'overview', label: 'Overview' },
  { id: 'acreage', label: 'Acreage' },
  { id: 'insurance', label: 'Insurance' },
  { id: 'claims', label: 'Claims' },
  { id: 'calendar', label: 'Calendar' },
]
```

Use `useSearchParams()` and `useRouter()` from `next/navigation`. Read `?tab=` param (default `'overview'`), `?farm=`, `?crop=` params.

Navigation helper: `navigateTab(tabId)` calls `router.replace` preserving existing `?farm=` and `?crop=` params while replacing `?tab=`.

Filter change helper: `setFilter(key, value)` calls `router.replace` preserving `?tab=` while updating the given param. Empty string removes the param.

Layout structure:
1. **Header bar**: `text-lg font-mono font-bold text-glomalin-text` — "Compliance Hub" + sublabel "FSA · Insurance · Claims" in `text-glomalin-muted text-sm`.
2. **Tab nav**: horizontal list of TABS, each a `<button>` with `px-4 py-2 text-xs font-mono` — active tab gets `text-glomalin-accent border-b-2 border-glomalin-accent`, inactive gets `text-glomalin-muted hover:text-glomalin-text`.
3. **Filter bar** (below tab nav, `border-t border-glomalin-border pt-3 pb-3 flex gap-4 items-center`):
   - Label: "Filter:" in `text-xs text-glomalin-muted font-mono`
   - Farm input: `<input type="text" placeholder="Farm..." value={farm} onChange={...}` with glomalin styling (`bg-glomalin-surface border border-glomalin-border text-glomalin-text text-xs font-mono px-2 py-1 rounded`)
