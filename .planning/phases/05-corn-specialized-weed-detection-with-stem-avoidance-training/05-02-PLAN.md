**utils/corn_detector.py:**

Create `CornDetector` class that mirrors `GreenOnGreen.inference()` return signature so `owl.py` hoot() loop requires minimal changes.

- Import `from ultralytics import YOLO` and `import cv2`
- Define `CORN_CLASS_ID = 0` (class 0 = corn, class 1 = weed, per data.yaml order from CONTEXT.md)
- Constructor: `__init__(self, model_path: str, confidence: float = 0.75)`
  - `self.model = YOLO(model_path)` — accepts .pt or .engine files
  - `self.confidence = confidence`
  - Per RESEARCH.md: default 0.75 confidence (discretion recommendation)
- Method: `inference(self, frame, confidence=None, **kwargs)` — **kwargs for forward compat
  - `conf = confidence or self.confidence`
  - `results = self.model.predict(frame, conf=conf, verbose=False)[0]`
  - Iterate `results.boxes`:
    - Extract `cls_id = int(box.cls)`, `score = float(box.conf)`, `x1,y1,x2,y2 = map(int, box.xyxy[0])`
    - Compute `bw, bh = x2-x1, y2-y1` and `cx, cy = x1+bw//2, y1+bh//2`
    - Label: `"corn"` if `cls_id == CORN_CLASS_ID` else `"weed"`
    - Append to `boxes` list: `{"bbox": (x1, y1, bw, bh), "label": label, "conf": score, "center": (cx, cy)}`
    - If label is "corn", append `[cx, cy]` to `corn_centers`
  - Build `weed_centres` list: `[[cx, cy] for box in boxes if box["label"] == "weed"]`
  - Draw annotations on frame using `results.plot()` for `image_out`
  - Return `(None, boxes, weed_centres, image_out)` — matches GreenOnGreen signature exactly
  - Also expose `self.corn_centers` as instance attribute after inference (needed by stem avoidance)
  - Also expose `self.all_detections` (the full boxes list with labels) for stem avoidance filtering

**utils/frame_quality.py:**

Create frame quality gate per RESEARCH.md Pattern 3.

- `def frame_quality_ok(frame, blur_threshold=100, brightness_min=40) -> tuple[bool, str]:`
  - Convert to grayscale: `gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)`
  - Compute Laplacian variance: `lap_var = cv2.Laplacian(gray, cv2.CV_64F).var()`
  - Compute mean brightness: `mean_bright = np.mean(gray)`
  - If `mean_bright < brightness_min`: return `(False, f"dark (mean={mean_bright:.1f})")`
  - If `lap_var < blur_threshold`: return `(False, f"blurry (laplacian_var={lap_var:.1f})")`
  - Otherwise: return `(True, "ok")`
- Import cv2 and numpy

Per locked decision: "Poor visibility (heavy shadow, dawn/dusk, lens obstruction): pause actuation and alert operator — no blind actuation." This function provides the signal; the hoot() integration in Plan 03 will act on it.

**utils/stem_avoidance.py:**

Create buffer zone geometry per RESEARCH.md Pattern 2 and locked decisions.

- `def pixels_per_inch(frame_width_px: int, fov_inches: float) -> float:`
  - Return `frame_width_px / fov_inches`
  - Docstring: explain that fov_inches must be measured for the specific camera mount height

- `def is_in_buffer(weed_center: tuple, corn_detections: list, buffer_radius_px: float) -> bool:`
  - `wx, wy = weed_center`
  - For each detection in corn_detections where `det["label"] == "corn"`:
    - `cx, cy = det["center"]`
    - If `math.hypot(wx - cx, wy - cy) <= buffer_radius_px`: return True
  - Return False

- `def filter_weeds_for_actuation(detections: list, buffer_radius_px: float) -> tuple[list, list]:`
  - Separate detections into corn_detections and weed_detections by label
  - For each weed detection, check `is_in_buffer(weed["center"], corn_detections, buffer_radius_px)`
  - Return `(actionable_weeds, suppressed_weeds)` — two lists
  - `actionable_weeds`: weed detections outside all corn buffer zones (safe to actuate)
  - `suppressed_weeds`: weed detections inside a buffer zone (skipped for crop protection)
  - Per locked decision: "Weeds detected inside the buffer zone are skipped entirely — crop protection always takes priority over weed kill rate"

Import `math` only.
