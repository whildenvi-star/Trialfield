<objective>
Add a breadcrumb navigation bar above portal iframe embeds showing the current module name and a persistent "Back to Dashboard" link.

Purpose: When users navigate to an embedded Express app (farm-budget, grain-tickets, etc.), they lose navigation context — no indication of where they are in the portal hierarchy and no obvious way back. This plan adds a slim breadcrumb bar between the portal header and the iframe.

Output: New EmbedBreadcrumb component, updated EmbedFrame layout, updated module page to pass module metadata.
</objective>
