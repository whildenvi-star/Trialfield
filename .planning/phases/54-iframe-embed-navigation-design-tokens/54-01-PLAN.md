<objective>
Unify color tokens across all 8 apps into a single authoritative block in platform-tokens.css so every app renders identical colors.

Purpose: Currently each app's style.css defines its own :root color block (--bg, --card, --primary, --text, --border, etc.). While the values happen to be identical today, they're maintained in 7+ separate locations. This plan extracts them into platform-tokens.css (which already exists in every app for text-scale tokens) and removes the duplicates from style.css, making platform-tokens.css the single source of truth for visual tokens.

Output: All 7 platform-tokens.css files updated with color token block; all 7 style.css/globals.css files cleaned of duplicate color definitions.
</objective>
