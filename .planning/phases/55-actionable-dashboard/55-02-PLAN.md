**Props:** `initial: ActionItemsResponse | null` (SSR pre-fetched data, same pattern as OfflineSummaryCards)

**State and fetching:**
- useState for `data: ActionItemsResponse | null` (initialized from `initial`)
- useState for `loading: boolean` (true on mount if no initial)
- useEffect on mount: fetch `/api/dashboard/action-items`, update state. Follow same offline/online pattern as OfflineSummaryCards (navigator.onLine checks, online/offline event listeners, re-fetch on reconnect)

**Layout per user decisions (from CONTEXT.md):**

1. **Total count header:** At top, "{N} items need attention" in text-glomalin-text font-mono. Only show when totalCount > 0.

2. **Module groups:** For each group in data.groups:
   - Group header row: clickable, navigates to that module's main page (use MODULES from modules.ts to find the route by matching module id). Show module label in text-glomalin-text font-mono font-bold, with a subtle right-arrow icon. If group.offline is true, show dimmed (opacity-50) with "(service offline)" text in text-glomalin-muted and an "Unavailable" tag.
   - Item rows: compact list rows (not cards). Each row contains:
     - Severity icon: warning = amber triangle (text-amber-400), info = blue dot (text-blue-400). Use inline SVG, not emoji — the checkmark SVG is the user's only emoji exception for empty state.
     - Summary text: `item.summary` in text-glomalin-text font-mono text-sm
     - Source badge: `[{group.badge}]` in a small pill — bg-glomalin-surface border border-glomalin-border text-glomalin-muted text-xs font-mono px-2 py-0.5 rounded. Badge colors per module within soil-dark palette (Claude's discretion): FSA = border-amber-700/40, INS = border-yellow-700/40, CLM = border-red-700/40, GT = border-glomalin-green/40, BUDG = border-blue-700/40
     - Age: `item.age` if present, in text-glomalin-muted text-xs
   - Clicking an item row: wraps in `<Link href={item.link}>` for standard Next.js navigation
   - Rows should have hover:bg-glomalin-surface/50 transition and cursor-pointer

3. **Empty state:** When totalCount is 0 and not loading:
   - Centered checkmark icon (SVG, glomalin-green) + "Nothing needs attention" text in text-glomalin-muted font-mono
   - No module links, no cards, no quick links (per user decision)

4. **Loading state:** Simple skeleton — 3 rows of pulsing bg-glomalin-surface rounded bars (animate-pulse)

**Key styling:**
- All within soil-dark palette (bg-glomalin-bg, surface, border, accent, text, muted, green)
- font-mono throughout
- No external UI libraries
