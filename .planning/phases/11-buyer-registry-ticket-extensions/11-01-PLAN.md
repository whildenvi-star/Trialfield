  <name>Task 1: Add Buyer CRUD and BuyerColumnMap API routes to server.js</name>
  <files>grain-tickets/server.js</files>
  <action>
Add 6 new route handlers to server.js, placed after the existing Farms CRUD section (after the `app.delete('/api/farms/:id', ...)` route) and before the `/api/farm-names` route:

**Buyer CRUD (4 routes):**

1. `GET /api/buyers` — `prisma.buyer.findMany({ orderBy: { name: 'asc' } })`. Returns JSON array of all buyers.

2. `POST /api/buyers` — Accept `{ name, type, shortCode, notes }` in req.body. Trim `name`, require it non-empty (400 if missing). Trim `shortCode` or null. `type` should be one of: "elevator", "maltster", "co-op", "other", or null. Create via `prisma.buyer.create()`. Return 201 with buyer JSON. Handle P2002 (name unique violation) with 409.

3. `PUT /api/buyers/:id` — Accept partial update of `name`, `type`, `shortCode`, `notes`. Parse `id` as int (404 if NaN). Update via `prisma.buyer.update()`. Handle P2025 (not found) with 404, P2002 with 409.

4. `DELETE /api/buyers/:id` — Parse `id` as int. Before deleting, count tickets referencing this buyer: `prisma.ticket.count({ where: { buyerId: id } })`. If count > 0, return 409 with message: `"Cannot delete — ${count} ticket(s) reference this buyer. Reassign them first."`. Otherwise delete via `prisma.buyer.delete()`. Handle P2025 with 404.

**BuyerColumnMap (2 routes):**

5. `GET /api/buyers/:id/column-maps` — `prisma.buyerColumnMap.findMany({ where: { buyerId } })`. Returns array of column mappings for the buyer.

6. `PUT /api/buyers/:id/column-maps` — Accept `{ fieldName, csvColumn }` in req.body. Both required (400 if missing). Upsert via `prisma.buyerColumnMap.upsert({ where: { buyerId_fieldName: { buyerId, fieldName } }, update: { csvColumn }, create: { buyerId, fieldName, csvColumn } })`. Note the compound unique index name `buyerId_fieldName` (Prisma auto-generates from `@@unique([buyerId, fieldName])`).

All routes follow the existing try/catch + console.error pattern from the Crops and Farms CRUD routes already in server.js.
