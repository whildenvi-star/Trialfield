Create `src/components/layout/mobile-bottom-nav.tsx` as a client component ('use client').

Props interface:
```ts
interface MobileBottomNavProps {
  grantedModuleIds: string[]
}
```

The protected layout already fetches `grantedModules` (array of Module objects). This component receives the granted module IDs as a string array so it can filter the "More" list.

**Main tab definitions** (locked decision from CONTEXT.md):
```ts
const MAIN_TABS = [
  { label: 'Home',         href: '/dashboard',           icon: 'home' },
  { label: 'Farm Info',    href: '/app/field-history',   icon: 'field' },
  { label: 'Field Passes', href: '/app/field-ops',       icon: 'check' },
  { label: 'More',         action: 'more' as const,      icon: 'grid' },
]
```

Per RESEARCH.md open question resolution: Farm Info = `/app/field-history`, Field Passes = `/app/field-ops` — these are Claude's discretion.

**SVG icons** (inline, no icon library per don't-hand-roll rules):
- Home: simple house outline SVG (24x24)
- Field: a grid/field outline SVG (24x24) — or use a leaf/plant icon for farm context
- Check: checkmark circle SVG (24x24)
- Grid: 3x3 grid dots SVG (24x24)

Use simple, recognizable inline SVGs consistent with the hamburger icon pattern in side-nav.tsx (viewBox="0 0 24 24", stroke-based, no fill).

**Tab bar structure:**
- `fixed bottom-0 inset-x-0 z-50`
- `bg-glomalin-bg border-t border-glomalin-border`
- `flex` with 4 flex-1 items
- Each item: `flex flex-col items-center justify-center flex-1 min-h-[56px] touch-manipulation`
- Active tab: `text-glomalin-accent` — use `usePathname()` to detect active route (`pathname.startsWith(tab.href)` for non-home tabs; exact match for `/dashboard`)
- Inactive tab: `text-glomalin-muted`
- Label: `text-[10px] font-mono mt-0.5`
- Icon: SVG at `w-6 h-6`

**"More" overflow sheet:**
- `useState(false)` for `sheetOpen`
- When `sheetOpen`:
  - Full-screen backdrop: `fixed inset-0 z-40 bg-black/50` with `onClick={() => setSheetOpen(false)}`
  - Sheet panel: `fixed inset-x-0 bottom-0 z-50 bg-glomalin-bg border-t border-glomalin-border rounded-t-xl`
  - Sheet slides in via `translate-y` transition: closed = `translate-y-full`, open = `translate-y-0`, with `transition-transform duration-200`
  - Sheet header: "More" title + close button (`×`)
  - Module list: Import `MODULES` from `@/lib/modules.ts`. Filter to modules where `mod.id` is in `grantedModuleIds` AND `mod.id` is NOT in MAIN_TABS hrefs (exclude home, field-history, field-ops). For each: `<Link href={/app/${mod.id}} onClick={() => setSheetOpen(false)} className="flex items-center gap-3 px-4 py-3 text-sm font-mono text-glomalin-text border-b border-glomalin-border last:border-0">{mod.label}</Link>`
  - Closing navigation: Link's onClick closes the sheet

**CSS variable on mount** (per Pitfall 6 in RESEARCH.md — belt-and-suspenders):
