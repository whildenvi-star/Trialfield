Check the supabase/migrations directory for the correct next migration number — if 005 is taken, use 006, etc.

Schema for `field_boundaries`:
```sql
CREATE TABLE IF NOT EXISTS field_boundaries (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  registry_field_id TEXT NOT NULL,
  name TEXT NOT NULL,
  geojson JSONB NOT NULL,
  centroid_lat DOUBLE PRECISION,
  centroid_lng DOUBLE PRECISION,
  imported_at TIMESTAMPTZ DEFAULT NOW(),
  CONSTRAINT field_boundaries_registry_field_id_unique UNIQUE (registry_field_id)
);
```

Schema for `farm_map_config` (key-value store for map settings like derived farm center):
```sql
CREATE TABLE IF NOT EXISTS farm_map_config (
  key TEXT PRIMARY KEY,
  value JSONB NOT NULL,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

RLS policies:
- `field_boundaries`: SELECT for authenticated users; INSERT/UPDATE for admin only (`auth.jwt() ->> 'role' = 'admin'` or via service role)
- `farm_map_config`: SELECT for authenticated users; INSERT/UPDATE for admin only

Enable RLS on both tables:
```sql
ALTER TABLE field_boundaries ENABLE ROW LEVEL SECURITY;
ALTER TABLE farm_map_config ENABLE ROW LEVEL SECURITY;
```

Add policies with names like `field_boundaries_select`, `field_boundaries_admin_write`, etc.

Note: The portal uses Supabase SSR auth (`@supabase/ssr`). The admin import API route will use the service role key to bypass RLS — that's fine. Public reads use the anon key with the SELECT policy.

Create `glomalin-portal/src/lib/map-config.ts` — this is the editable config file the user can tune without a redeploy.

**Library decision:** Use MapLibre GL JS (`maplibre-gl` npm package) — it is the open-source fork of Mapbox GL JS with an identical API surface. No proprietary token required. Satellite tiles are served via MapTiler Cloud (free tier requires a free API key, noted below) or Stamen/USGS as a fallback.

**MapTiler note:** MapTiler Cloud free tier requires a free API key (register at maptiler.com). The key is not secret — it is a public key suitable for `NEXT_PUBLIC_MAPTILER_KEY`. If the user does not set this key, use the Stamen satellite fallback URL which requires no key.

Contents:

```typescript
// map-config.ts
// Edit crop colors here to tune the field map palette without code changes.
// Crop names must match canonical names from the crop registry (Phase 50).

export const CROP_COLORS: Record<string, string> = {
  // Grain crops
  'Yellow Corn': '#F5C518',           // corn = golden yellow
  'Soybeans': '#6B8E23',              // soybeans = olive green
  'Soft Red Winter Wheat': '#D4A017', // winter wheat = amber
  'Hard Red Winter Wheat': '#C8860A', // HRW wheat = darker amber (matches portal accent)
  'Hybrid Rye': '#7B5EA7',            // rye = purple
  'Oats': '#C8A97E',                  // oats = tan
  'Barley': '#A89060',                // barley = warm tan
  // Specialty / organic
  'Natto Beans': '#8BAF5A',           // natto = lighter green
  'Seed Beans': '#4A7C59',            // seed beans = forest green
  // Fallback for any crop not listed above
  '__unassigned': '#4A4035',          // no crop = muted dark (dark soil neutral)
  '__unknown': '#3A3028',             // crop set but not in palette = darker muted
}

// Opacity for the polygon fill (0-1). Borders are always full opacity.
export const FILL_OPACITY = 0.55

// Opacity on hover — lighter to signal clickability
export const HOVER_FILL_OPACITY = 0.80

// Dashed line pattern for organic fields overlay (MapLibre line-dasharray)
// [dash length, gap length] in pixels at zoom ~14
export const ORGANIC_DASH_PATTERN = [3, 2]

// Organic border color (white-ish so it contrasts with any crop fill)
export const ORGANIC_BORDER_COLOR = '#E8D8C0'
export const ORGANIC_BORDER_WIDTH = 2.5

// Standard polygon border
export const STANDARD_BORDER_COLOR = '#2A2218'
export const STANDARD_BORDER_WIDTH = 1.5

// Default map center used as a loading fallback only.
// At runtime, fitBounds() on all field polygons is the canonical initial view.
export const DEFAULT_MAP_CENTER: [number, number] = [-92.5, 41.9] // [lng, lat]
export const DEFAULT_MAP_ZOOM = 13

// Satellite tile style URL for MapLibre GL JS.
// Uses MapTiler Cloud satellite (free tier, requires NEXT_PUBLIC_MAPTILER_KEY).
// Falls back to a no-key-required source if env var is absent.
export function getSatelliteStyleUrl(): string {
  const maptilerKey = process.env.NEXT_PUBLIC_MAPTILER_KEY
  if (maptilerKey) {
    return `https://api.maptiler.com/maps/satellite/style.json?key=${maptilerKey}`
  }
  // Fallback: USGS satellite via a public MapLibre-compatible style
  // (lower quality but requires no account)
  return 'https://demotiles.maplibre.org/style.json'
}
```

This file is the single source of truth for all color/style values. The map component imports from here — nothing is hardcoded in component files.

Note: There is NO hybrid style, NO layer switcher, and NO Mapbox style URIs. Satellite imagery only per locked CONTEXT.md decision. The `MAPBOX_STYLES` export from earlier drafts is removed.
