Import maplibre-gl dynamically inside useEffect (not at module level) to avoid SSR issues:
```typescript
const maplibregl = await import('maplibre-gl')
await import('maplibre-gl/dist/maplibre-gl.css')
```

2. **Map initialization** via `useEffect` on mount:
   - `new maplibregl.Map({ container: mapRef.current, style: getSatelliteStyleUrl(), center: DEFAULT_MAP_CENTER, zoom: DEFAULT_MAP_ZOOM })`
   - Note: DEFAULT_MAP_CENTER is a loading fallback only. The canonical initial view is set by fitBounds after polygons load.
   - On `map.on('load')`: fetch `/api/maps/boundaries`, then call `initializeLayers(map, featureCollection)`

3. **Auto-zoom to fit all fields (fitBounds — REQUIRED per CONTEXT.md):**

After fetching boundaries and before calling initializeLayers, compute the bounding box of ALL field polygons and call fitBounds:

```typescript
// Compute bbox from all features
let minLng = Infinity, maxLng = -Infinity, minLat = Infinity, maxLat = -Infinity
for (const feature of featureCollection.features) {
  const coords = flattenCoordinates(feature.geometry)
  for (const [lng, lat] of coords) {
    if (lng < minLng) minLng = lng
    if (lng > maxLng) maxLng = lng
    if (lat < minLat) minLat = lat
    if (lat > maxLat) maxLat = lat
  }
}

if (isFinite(minLng)) {
  map.fitBounds([[minLng, minLat], [maxLng, maxLat]], { padding: 40, animate: false })
}
```

`flattenCoordinates(geometry)` is a helper that handles Polygon (outer ring only) and MultiPolygon (all outer rings). Returns an array of [lng, lat] pairs.

4. **Fetch registry data for crop/organic:**

The boundaries route only returns `{ registry_field_id, name, centroid_lat, centroid_lng }`. Crop and organic status must come from farm-registry. Make a second fetch inside the `map.on('load')` callback:

```typescript
const [boundaryRes, metaRes] = await Promise.all([
  fetch('/api/maps/boundaries'),
  fetch('/api/maps/fields-meta')  // new lightweight route — see note below
])
```

If `/api/maps/fields-meta` route does not exist yet, inline the farm-registry fetch inside the boundaries GET route (update Plan 02's route to join registry data server-side). Either approach is acceptable — choose whichever minimizes new files. Document which approach was taken in SUMMARY.md.

Simplest approach: Update `/api/maps/boundaries/route.ts` to also call `fetchRegistryService('/api/fields')` and merge `{ crop, organic, reportingAcres }` into each feature's properties by matching `registry_field_id` to the registry field `id`. Then FieldMap only needs one fetch.

Feature properties available to layers: `{ registry_field_id, name, centroid_lat, centroid_lng, crop, organic, reportingAcres }`.

5. **Layer setup** (`initializeLayers` function):
   - Add source: `map.addSource('fields', { type: 'geojson', data: featureCollection })`
   - Build MapLibre match expression for fill color from CROP_COLORS:
     ```typescript
     const colorExpression: maplibregl.ExpressionSpecification = [
       'match', ['get', 'crop'],
       ...Object.entries(CROP_COLORS)
         .filter(([k]) => !k.startsWith('__'))
         .flatMap(([crop, color]) => [crop, color]),
       CROP_COLORS.__unassigned  // fallback
     ]
     ```
   - Add fill layer with color expression and FILL_OPACITY
   - Add standard border layer using STANDARD_BORDER_COLOR, STANDARD_BORDER_WIDTH
   - Add organic dashed border layer filtered to `['==', ['get', 'organic'], true]` using ORGANIC_BORDER_COLOR, ORGANIC_BORDER_WIDTH, ORGANIC_DASH_PATTERN
   - Add hover fill layer using feature-state `hover`:
     ```
     'fill-opacity': ['case', ['boolean', ['feature-state', 'hover'], false], HOVER_FILL_OPACITY, 0]
     ```

6. **Hover interaction:**
   - `map.on('mousemove', 'fields-fill', (e) => { ... })` — set feature-state hover=true on hovered feature, clear previous
   - Show popup with field name only: `new maplibregl.Popup({ closeButton: false, closeOnClick: false }).setLngLat(e.lngLat).setHTML('<span>' + name + '</span>').addTo(map)`
   - Popup style via CSS: monospace, dark bg `#0e0c0b`, border `#2a2218`, text `#e8d8c0`
   - `map.on('mouseleave', 'fields-fill', () => { clear hover state; popup.remove() })`

7. **Click interaction:**
   - `map.on('click', 'fields-fill', (e) => { setSelectedField(feature.properties) })`
   - `selectedField` state controls FieldDetailPanel visibility

8. **NO layer switcher:** Do not add any style toggle buttons, setStyle calls, or basemap controls. Satellite imagery only per CONTEXT.md locked decision.

9. **Legend:** Render `<MapLegend crops={activeCrops} />` as a React child overlaid on the map container. Position with absolute/fixed CSS (handled inside MapLegend).

10. **Cleanup:** Return cleanup function from useEffect that calls `map.remove()`
